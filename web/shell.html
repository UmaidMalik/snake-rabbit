Snake & Rabbit

A classic snake game with a wild rabbit that moves around the board when you get close. GitHub

Written in C using raylib and compiled for the browser with WebAssembly.

Loading WebAssembly game...

How to Play

  • Use the Arrow Keys or WASD to move.
  • Use Spacebar for a speed boost.
  • Eat the rabbit to grow and earn points.
  • You can pass through walls, but not your own body.

Controls

Key Action
Arrow Keys / WASD Move the snake
Spacebar Speed boost
P Pause or resume
R Restart

Scoring

  • Earn 1 point for every rabbit eaten.
  • The snake grows after each rabbit.
  • Survive as long as possible.

Technology

  • Game logic written in C
  • Rendering and input powered by raylib
  • Compiled to WebAssembly with Emscripten
  • Runs directly in the browser
